# Markdown 프린터

Markdown 파일을 쉽게 프린트할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- **파일 업로드**: .md, .markdown, .txt 파일 업로드 지원
- **직접 입력**: 텍스트 에디터로 Markdown 직접 작성 가능
- **드래그 앤 드롭**: 파일을 끌어다 놓기로 간편하게 업로드
- **실시간 미리보기**: Markdown을 HTML로 변환하여 미리보기
- **프린트 최적화**: 프린트에 최적화된 레이아웃
- **키보드 단축키**: 빠른 작업을 위한 단축키 지원

## 사용 방법

1. 웹 브라우저에서 `index.html` 파일을 엽니다
2. Markdown 파일을 업로드하거나 직접 입력합니다
3. "미리보기" 버튼으로 결과를 확인합니다
4. "프린트" 버튼으로 문서를 출력합니다

## 키보드 단축키

- `Ctrl/Cmd + P`: 프린트
- `Ctrl/Cmd + Enter`: 미리보기

## 기술 스택

- HTML5
- CSS3
- JavaScript (ES6+)
- [Marked.js](https://marked.js.org/) - Markdown 파싱
